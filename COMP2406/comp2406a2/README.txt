Start server using (node server.js). Start on localhost:3000 on mainmenu, click on join game which will direct to game page. Enter a name and click join. Game will begin when one player joins. Only proceeds to next question once everyone has answered. When game ends, all users are disconnected and will need to rejoin if they want to play again.

Add-On Features:
- Added a statistics page and stores the statistics into a file
- Chat room when player joins the game (only for players in game)

Modules:
- Express
- Socket.io
- Request

npm install express
npm install socket.io
npm install request
